# 小手机产品文档

这组文档面向混合读者，目标是把 `generic-mini-phone` 从当前原型整理成可持续推进的产品定义。

文档阅读顺序建议如下：

1. [产品总览](./product-overview.md)
2. [信息架构与对象模型](./information-architecture.md)
3. [核心流程](./core-flows.md)
4. [版本路线图](./roadmap.md)

当前文档默认基线：

- 当前实现载体：`generic-mini-phone`
- 核心参考：`SillyTavern`
- 未来能力参考：`opencode` 与自有 agent 体系
- 当前目标版本：产品 `v1`

当前仓库现状：

- 已有纯前端手机壳与内容原型
- 已有角色、记忆、世界书、聊天、导入、设置等基础模块
- 已支持 OpenAI 兼容接口直连与前端 prompt 预览
- 仍未引入正式 agent runtime、工具编排和后端会话层
